# Boilerplate32

My boilerplate of HTML/CSS.


## Naming conventions

### Serial Number

 1. alpha
 2. beta
 3. gamma
 4. delta
 5. epsilon
 6. zeta

* * *

 1. primary
 2. secondary
 3. tertiary

### Prefix

 + __hd__ - header
 + __bd__ - body
 + __ft__ - footer
 + __line__ - line/rule/hr
 + __tb__ - tab
 + __bt__ - button
 + __hl__ - headline text
 + __bx__ - box/section
 + __bg__ - background
 + __fig__ - figure
 + __bar__ - bar
 + __bnr__ - banner
 + __nav__ - navigation
 + __mod__ - module

### States

 + on
 + off
 + active
 + disabled
 + hidden
