# Maple.css



v0.1

## Conventions

### Lint

 + Disallow IDs in selectors
 + Disallow tags only in selectors
 + Disallow units for 0 values
 + Require shorthand properties


### Comments
```
//-------------------------------------
//  First Level
//-------------------------------------

// ----- Second Level ----- //

// Third Level
```

### Serial Number

 1. primary
 2. secondary
 3. tertiary

* * *

 1. a
 2. b
 3. c
 4. d
 5. e
 6. f

* * *

 1. alpha
 2. beta
 3. gamma
 4. delta
 5. epsilon
 6. zeta

### Prefix

 + __hd-*__ - header
 + __bd-*__ - body
 + __ft-*__ - footer
 + __hl-*__ - headline text
 + __bx-*__ - box/section
 + __bg-*__ - background
 + __btn-*__ - button
 + __fig-*__ - figure
 + __icn-*__ - icon
 + __tab-*__ - tab
 + __bar-*__ - bar
 + __bnr-*__ - banner
 + __nav-*__ - navigation
 + __mod-*__ - module
 + __line-*__ - line/rule/hr

### States

 + is-on
 + is-off
 + is-active (is-selected)
 + is-pressed (:hover.:avtive)
 + is-animated
 + is-disabled
 + is-hidden (display:none)
 + is-collapsed
