# Boilerplate32

My boilerplate of HTML/CSS.


## Naming conventions

### Serial Number

 1. alpha
 2. beta
 3. gamma
 4. delta
 5. epsilon
 6. zeta

* * *

 1. primary
 2. secondary
 3. tertiary

### Prefix

 + __hd - header
 + __bd - body
 + __ft - footer
 + __line - line/rule/hr
 + __tb - tab
 + __bt - button
 + __hl - headline text
 + __bx - box/section
 + __bg - background
 + __fig - figure
 + __bar - bar
 + __bnr - banner
 + __nav - navigation
 + __mod - module

### States

 + on
 + off
 + active
 + disabled
 + hidden
